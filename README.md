### My Analysis Approach

1. **Discover** — understand the business problem, users, constraints and desired outcomes.
2. **Define** — translate needs into clear requirements, KPIs and acceptance criteria.
3. **Analyse** — examine processes, data, risks, root causes and performance gaps.
4. **Design** — build practical models, dashboards, workflows and recommendations.
5. **Validate** — reconcile results, test usability and confirm stakeholder expectations.
6. **Improve** — monitor outcomes and refine the solution using evidence.
